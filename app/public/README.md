# Public Folder - Apollo Hospital Logo

## Place Your Apollo Hospital Logo Here

1. Save your Apollo Hospital logo image as: `apollo-logo.png`
2. Place it in this `public/` folder
3. The logo will be automatically displayed on all pages

## Supported Formats:
- PNG (recommended)
- SVG
- JPG/JPEG

## Recommended Size:
- 200x200 pixels or larger
- Transparent background (PNG) works best

## File Location:
```
app/
└── public/
    └── apollo-logo.png    ← Place your logo here
```

The logo will be accessible at: `/apollo-logo.png`
